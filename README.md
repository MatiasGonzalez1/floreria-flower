# Florería Flower

Página estática de presentación para Florería Flower, una florería premium en Posadas, Misiones.

## Descripción

Este proyecto es un sitio de landing page hecho en HTML/CSS/JavaScript para mostrar los servicios de una florería local, su trayectoria, servicios principales, galería de arreglos, testimonios y datos de contacto.

Cuenta con:

- Hero section con llamada a la acción por WhatsApp.
- Navegación fija con scroll suave a secciones.
- Sección de historia y experiencia (40+ años).
- Servicios destacados con precios de referencia.
- Galería de arreglos florales.
- Testimonios de clientes.
- FAQ y contacto directo.
- CTA fijo en móviles y botón flotante de WhatsApp.

## Tecnologías

- HTML5
- Tailwind CSS (CDN)
- JavaScript para animaciones simples y comportamiento interactivo
- Lucide Icons (CDN)

## Archivos

- `index.html` — página principal del sitio.

## Cómo usar

1. Abre `index.html` en tu navegador.
2. O bien, sirve el proyecto localmente con un servidor estático si quieres probar la navegación sin restricciones de archivos locales.

Ejemplo usando Python:

```bash
python -m http.server 8000
```

Luego abre `http://localhost:8000`.

## Personalización

- Cambia textos directamente en `index.html`.
- Reemplaza las imágenes con URLs propias o archivos locales.
- Ajusta los enlaces de WhatsApp al número y mensaje que prefieras.
